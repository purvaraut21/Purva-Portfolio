# 🌐 Portfolio Project Documentation

## 📌 Project Overview

This is my personal portfolio website.
It shows my skills, projects, and experience as a developer.
The website is simple, interactive, and easy to use.

---

## 🎯 Objective

* To show my projects and skills clearly
* To create a professional online profile
* To help recruiters understand my work
* To practice frontend development

---

## 🛠️ Technologies Used

* HTML – for structure
* CSS (Tailwind CSS) – for design and layout
* JavaScript – for animations and interactivity

---

## ✨ Features

### 1. Hero Section

* Shows my name and role
* Typing animation (different roles appear automatically)
* Buttons to download resume and certificates

---

### 2. About Me

* Short introduction about me
* My interests and current work

---

### 3. Stats Section

* Shows numbers like:

  * Projects completed
  * Years of study
  * Coding hours
* Numbers increase with animation

---

### 4. Education & Experience

* Shows my education (BE Computer Engineering)
* Shows my experience:

  * Trainee Developer at Writer Pvt Ltd (2022–2023)

---

### 5. Skills Section

* Shows my skills using progress bars:

  * Java
  * Python
  * React
  * Django

---

### 6. Projects Section

Shows all my projects with:

* Project name
* Description
* GitHub code link
* Live/demo status

#### Main Projects:

* Banking System (Core Java)
* Meme Generator (React)
* Music Podcast Player
* Job Portal (Django + React – in progress)

---

### 7. Contact Section

* Contact form (user can send message)
* Links to:

  * Email
  * LinkedIn
  * GitHub

---

### 8. Dark Mode / Light Mode

* User can switch theme
* Theme is saved automatically

---

### 9. Animations

* Smooth scrolling
* Fade-in on scroll
* Hover effects on cards
* Typing animation

---

## 📂 Project Structure

```bash
portfolio/

```
